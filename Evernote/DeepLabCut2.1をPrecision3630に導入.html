Here is an example on how to install the GPU driver + CUDA 9 + optional TensorFlow 1.12 will follow here:
FIRST, install a driver for your GPU (we recommend the 384.xx) Find DRIVER HERE: https://www.nvidia.com/download/index.aspx
ーー＞Quadro RTX 4000
　　　・Windows Driver Type:　でDCHを選ばないとエラーが出た


SECOND, install CUDA (9.0 here): https://developer.nvidia.com/cuda-90-download-archive (Note that cuDNN, https://developer.nvidia.com/cudnn, is supplied inside the anaconda environment files, so you don't need to install it again).

ーー＞CUDA9、Base installからPatch4まで
　　　・Windows Driver Type:　でDCHを選ばないとエラーが出た

　　　・Visual Studioが必要との警告

Step 3: Easy install for Windows
https://github.com/AlexEMG/DeepLabCut/blob/master/conda-environments/README.md

ファイルをダウンロードして、
conda env create -f dlc-windowsGPU.yaml
を実行

activate dlc-windowsGPU
を実行

THIRD: Launch the provided Anaconda Environment File (see here!) or [OPTIONAL] install TensorFlow independently:
Package for pip install:
pip install tensorflow-gpu==1.12 —with GPU support (Ubuntu and Windows)
Note, the version is specified by using: ==1.12

（B) CUDA 10 を入れる場合ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー

環境に自分で名前を付ける場合：
conda env create -n dlc -f dlc-windowsGPU.yaml

環境を削除する場合：
conda remove -n dlc-windowsGPU --all
